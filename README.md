# 📚 Fundamentos Java - DIO

Repositório com exercícios práticos desenvolvidos durante meus estudos de Java pela DIO.

## 🧠 Conceitos aplicados

- Entrada de dados com Scanner
- Tipos primitivos (int, double)
- Cálculos matemáticos
- Formatação de saída (printf)
- Tratamento de entrada com Locale.US
- Manipulação de strings
- controle de fluxo básico

## 📌 Exercícios

### ✔ Cálculo de idade
Recebe nome e ano de nascimento e calcula a idade.

### ✔ Área do quadrado
Calcula a área a partir do lado informado.

### ✔ Área do retângulo
Calcula a área com base e altura.  
Utiliza padrão internacional (Locale.US) para leitura de valores decimais.

### ✔ Diferença de idade
Calcula a diferença de idade entre duas pessoas.  
Inclui tratamento de entrada para evitar problemas com buffer do Scanner.
