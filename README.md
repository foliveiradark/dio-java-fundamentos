# 📚 Fundamentos Java - DIO

Este repositório contém exercícios práticos desenvolvidos durante meus estudos de Java pela DIO.

## 🧠 Conteúdos abordados

- Entrada de dados com Scanner
- Cálculos básicos
- Lógica de programação

## 📌 Exercícios

### ✔ Cálculo de idade
Programa que recebe nome e ano de nascimento e calcula a idade.

### ✔ Área do quadrado
Programa que calcula a área de um quadrado a partir do valor do lado.

### ✔ Área do retângulo
Programa que calcula a área de um retângulo a partir da base e altura.

Utiliza padrão internacional (Locale.US) para leitura de valores decimais.

### ✔ Diferença de idade
Programa que calcula a diferença de idade entre duas pessoas.

Inclui tratamento de entrada para evitar problemas com leitura de dados (buffer do Scanner).
